# 🍔 Landing Page - Hamburgueria

Este projeto consiste em uma landing page desenvolvida para uma hamburgueria, com foco em apresentação visual, atração de clientes e experiência do usuário.

## 🚀 Funcionalidades

- Apresentação da marca
- Destaque de produtos (hambúrgueres)
- Seção principal (hero) com chamada atrativa
- Botão de ação (CTA - "Fazer Pedido")
- Layout moderno e organizado

## 💻 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 🎯 Objetivo do projeto

O objetivo deste projeto foi praticar a criação de interfaces modernas e responsivas, além de melhorar habilidades em design e estruturação de páginas web.

## 📚 Aprendizados

Durante o desenvolvimento, pratiquei:

- Estruturação de páginas com HTML
- Estilização com CSS (cores, layout, responsividade)
- Manipulação básica com JavaScript
- Organização de código front-end
- Criação de uma experiência visual atrativa

## ⚠️ Status do projeto

🚧 Em desenvolvimento

Melhorias futuras:
- Responsividade completa para dispositivos móveis
- Animações e interações mais avançadas
- Integração com sistema de pedidos
- Melhor organização do código
